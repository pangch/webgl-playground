WebGL Playground

A few demo applications built while learning WebGL.

  • First Person View

    A first person shooter like 3D scene generated from a 2D map. Implemented an ad hoc lighting model to create a spot light effect.

  • Text Engraving

    Engrave text on a prism solid. A basic CSG library is implemented based on BSP tree, which supports union, substraction and intersection of primitives.

  • Physics Simulation on GPU

    Simulation of a group of small balls spawned in a confined space, bouncing and colliding with each other. This demo was created to explore physics simulation and GPGPU techniques.

All demos are built with WebGL and gl-matrix library only. © Chen Pang, 2014. All rights reserved.
